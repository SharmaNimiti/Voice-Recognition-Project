# Voice-Recognition-Project
This project is a Python-based Voice Assistant integrated with a simple voice authentication system. It uses speech recognition and text-to-speech technologies to interact with users and perform various tasks such as searching information, opening websites, sending emails, and capturing images.

🛠️ Technologies Used
Python
SpeechRecognition
pyttsx3 (Text-to-Speech Engine)
Wikipedia API
smtplib (Email Service)
OpenCV (via ecapture)
Webbrowser Module

⚙️ How It Works
The system starts with voice authentication
User speaks a passphrase
If authenticated, the assistant is activated
The assistant:
Listens to commands
Converts speech to text
Processes the query
Responds using voice output
